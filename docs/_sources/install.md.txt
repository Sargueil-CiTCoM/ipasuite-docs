# Installation
1. On debian, install libdb-dev `sudo apt install libdb-dev`
3. Install and setup Miniconda : https://docs.conda.io/en/latest/miniconda.html
4. Create a conda environment for QuShape and install it : `conda create --name qushape -c free -c sargueil-citcom qushape`
5. (recommended) Create a conda environnent for rnasique `conda create --name rnasique -c conda-forge
   -c bioconda` and activate your environment `conda activate rnasique` 
4. Install RNASique `conda install -c conda-forge -c bioconda rnasique`
